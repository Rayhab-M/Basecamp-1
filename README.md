Installation

To install and run the project, follow these steps: clone the gitea or download the project files to your locally env and make sure rails, ruby, node.js and yarn has been installed on your device. to confirm this you can use the following command rails -v, ruby -v, node -v and yarn -v to be sure. after that run this command in your terminal to push all pending migration: rails db:migrate.

Usage

After the installation run this command rails s to start the server and follow the provided link localhost:3000 to enjoy this amazing work
